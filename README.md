# ID3DecisionTree
An ID3 DecisionTree for information theory course
## 测试项目
### 单树
#### 预剪枝
- 深度：[2-10] 多个数据集
- 节点：[1-30] 多个数据集
#### 优化
- 无深度限制，无节点尺寸限制，多个数据集
#### 得分函数
- IG：不带优化/带优化
- IGR：不带优化/带优化
### 森林
#### Boosting
- 树的深度限制：多个数据集，5棵树
- 树的数量：较好的深度限制，多个数据集
- 优化：较好的深度限制，较好的树的数量，多个数据集
#### Bagging
- 树的深度限制：多个数据集，5棵树
- 树的数量：较好的深度限制，多个数据集
- 优化：较好的深度限制，较好的树的数量，多个数据集
